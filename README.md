<h1>osTicket Ticket Lifecycle: Intake Through Resolution</h1>

<p>
Demonstrated the complete <strong>ticket lifecycle workflow</strong> within the osTicket help desk system, from ticket intake through resolution.  
This project showcases real-world IT support processes including ticket triage, assignment, internal communication, SLA application, and issue resolution.
</p>

<h2>Project Overview</h2>

<ul>
  <li>Created and submitted support tickets as an end user</li>
  <li>Triaged and assigned tickets as an agent</li>
  <li>Applied SLA priorities based on issue severity</li>
  <li>Documented internal notes and agent communication</li>
  <li>Worked tickets through resolution and closure</li>
</ul>
